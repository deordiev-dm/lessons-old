Плавні переходи. CSS властивість transition

Плавні переходи дозволяють поліпшити нашу верстку та зробити її більш живою та елегантною
Об'єкт без властивості transition змінюється різко, що не приємно для користувача
Властивіть transition-duration: 1000ms;
Усі властивості змінюються плавно за цей час
transition-property: transform;
Завдяки цій властивості ми можемо обирати які властивості змінювати плавно. В цьому прикладі border-raduis та background-color змінюються моментально.
transition-delay: 300ms;
Затримка відтворення переходу
transition-time-function: ease;
Значення за замовченням. Властивіть впливає за те, як змінюються значення з часом
transition-time-function: linear;
transition-timing-function: ease-in;
transition-timing-function: ease-out;
transition-timing-function: ease-in-out;
transition-timing-function: cubic-bezier(0.78, -0.17, 0.53, 1.51);
transition: border-radius, transform, background-color, 1000ms cubic-bezier(0.19, 0.53, 0.64, 0.86) 200ms;
Універсальна властивсть. Порядок запису: property duration timing-function delay
Приклад з зображенням
Можливо вказати різну поведінку для об'єкту під час наведенні мишою та після її відведення.

Анімація. CSS властивість animation

Анімація - це дуже потужний інструмент, який дозволяє зробити верстку ще більш живою. Анімовані об'єкти виглядають дуже круто, лише тоді, коли вони доречні, тож важливим аспектом є не перенеситити ними свою сторінку
