Виконав Деордієв Дмитро

Трансформування об'єктів допомагає вирішувати багато дизайнерських задач. Від позиціонування об'єктів до створення крутих анімацій

Властивіть transform зі значенням translate переносить об'єкт вздовж вісей X, Y

transform: translate(50px);

Зсув ніяк не впливає на оточення

transform: translate(10px, -20px);

Елемент що йде нижче по коду буде ближчим до користувача по вісі Z (буде накривати собою інші об'єкти)

transform: translate(-5%, 25%);

Відсоток береться від розмірів об'єкту, а не від розмірів батьківського елементу

transform: translateX(50px);
transform: translateY(20px);

Властивіть transform зі значенням scale масштабує об'єкти

transform: scale(1.1);

Функція приймає значення коефецієнту на який множиться вектор (прям як у шкільній програмі)

transform: scale(1.1, 1.5);

Помітьте як даний об'єкт накриває собою попередній, але ховається під наступний

transform: scale(20px);

Ні, пікселі не працюють

transform: scale(115%);

А відсотки - так! Хоча валідатор свариться, тож краще так не робити!

transform: scaleX(1.3);
transform: scaleY(1.5);
transform: scale(0.9);

При множенні вектора на число менше одиниці - його довжина зменшується, а напрям не змінюється

transform: scale(0);

Доволі логічно

transform: scale(-0.5);

При множенні вектора на від'ємне число він змінює свій напрям. При множенні вектора на число менше одиниці - його довжина зменшується

transform: scale(-1, 1);
transform: scale(1, -1);

Властивіть transform зі значенням rotate обертає об'єкт

transform: rotate(45deg);

Додатні кути обертають об'єкт за годинниковою стрілкою

transform: rotate(-45deg);

Від'ємні кути обертають об'єкт проти годинникової стрілки

transform: rotateX(60deg);

Об'єкт можна обертати навколо координатних вісей. Але це працює ліпше з 3D-трансформаціями (ничже)

transform: rotateY(60deg);

Зараз об'єкт обертається навколо вісі Y, але ми бачимо лише пласку проекцію, тому що зараз не ввімкнена глибина площини z

transform: rotate(200px);

Так як це оберт, то вказувати треба кути, а не пікселі чи будь-які інші одиниці вимірювання довжини

transform: rotate(2rad);

Наприклад, можна вказати такі одиниці вимірювання кутів як радіани чи гради

transform: rotate(-100grad);

100 градів = 90 градусів

transform: rotate(-180deg);

Властивіть transform зі значенням skew нахиляє об'єкт

transform: skew(15deg);
transform: skew(20deg, 0deg);
transform: skew(0deg, 20deg);
transform: skew(-15deg, 20deg);

Для більш складних анімацій та ефектів можна поєднувати декілька 2D-трансформацій для одного об'єкту

transform: translate(-100px, 10px) rotate(12deg);

skew(20deg) scale(-1, 1) rotate(20deg);

scale(1.1) translate(50px, 0px);

scale(0.99) translate(0px, -2px) rotate(1deg);

rotate(360deg) scale(0.5);

Властивіть transform-origin впливає на те, від якої точки починається трансформація.

rotate(30deg);

Без використання transform-origin. За замовченням значення дорівнює 50% 50%

rotate(30deg); transform-origin: left top;
transform: scale(0.7);
transform-origin: -30% 150%; transform: scale(0.7);
transform: translate(20%, 10%);
transform: translate(20%, 10%); transform-origin: 10px 100px;

Нема різниці яку точку об'єкта переміщати, бо при паралельному перенесенні всі точки зміщуються на однаковий вектор.

transform: skew(-10deg);
transform: skew(-10deg); transform-origin: 0% 0%;

Окрім пласкої 2d трансформації в нашому арсеналі наявні інструменти для створення об'ємних перетворень об'єктів

rotateX(45deg);

Оберт навколо вісі Х при пласкій трансформації

rotateX(45deg); perspective: 400px;

Задамо БАТЬКІСЬКОМУ блоку властивість perspective, яка встановлює відстань між площиною z=0 та "користувачем"

rotateY(45deg);

Оберт навколо вісі Y при пласкій трансформації

rotateY(45deg); perspective: 400px;

Задамо БАТЬКІСЬКОМУ блоку властивість perspective, яка встановлює відстань між площиною z=0 та "користувачем"

rotateY(45deg); perspective: 400px; perspective-origin: 100% 100%

Задамо БАТЬКІСЬКОМУ блоку властивість perspective-origin, яка встановлює точку з якої дивиться користувач. За замовченням - центр об'єкту (50% 50%)

transform: translate3D(10px, -20px, 50px); perspective: 400px;

Переносить на 50px вгору по вісі z, що для користувача просто збільшує об'єкт

transform: translate3D(30%, 10%, 20%); perspective: 400px;

Вісь z не працює з відсотками, бо товщина об'єкту нульова

transform: translate3D(30%, 10%, 20px); perspective: 400px;

Можна вказати значення в пікселях тільки для вісі z

transform: rotate3D(1, 1, 1, 50deg); perspective: 400px;

Вказуємо координати вектора, а потім кут оберту

transform: rotate3D(2, 0, -1, 30deg); perspective: 400px;

Вказуємо координати вектора, а потім кут оберту

transform: scale3D(1.5, 1, 2) rotate3D(1, 0, 0, 40deg); perspective: 400px;
transform: rotate3D(1, 0, 0, 45deg); perspective: 400px; transform-style: flat;
transform: rotate3D(1, 0, 0, 45deg); perspective: 400px; transform-style: preserve-3d;
transform: rotate3d(0, 1, 0, 180deg); perspective: 400px; backface-visibility: visible;
transform: rotate3d(0, 1, 0, 180deg); perspective: 400px; backface-visibility: hidden;

Священний контрнаступ не зупинити
